# Eaglercraft Alpha 1.2.6

![preview](https://raw.githubusercontent.com/EaglerPorts/a1.2.6/refs/heads/main/img/mainmenu.png)
:-:
This is a fully functional web port of Minecraft Alpha 1.2.6, playable directly in any modern web browser. It currently supports Singleplayer with some buggy Multiplayer testing.

> [!WARNING]
> This is not a pirated version of Minecraft, This is a project that simply is just some "addons" for Minecraft Alpha 1.2.6 that allows it to be playable in the browser, This project does not distribute or include any copyrighted assets from the original game.

## Quick Start
### Play Now:
**Client:** https://eaglerports.github.io/a1.2.6/

**Other Minecraft Ports:** https://eaglerports.github.io

### Offline Downloads:
You can grab the latest builds here: [Releases](https://github.com/EaglerPorts/a1.2.6/releases) or compile a offline build yourself.

## Compiling
The files required for compiling can be found inside of the ```target_teavm_wasm_gc``` folder and there is another folder called ```buildtools``` which contains Java files to help you in the process. The final assets.epw and offline download will be available for you in the `javascript_dist` folder.
