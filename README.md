# Eaglercraft Alpha 1.2.6

![preview](https://raw.githubusercontent.com/EaglerPorts/a1.2.6/refs/heads/main/img/mainmenu.png)
:-:
This is a fully functional web port of Minecraft Alpha 1.2.6, playable in any modern web browser. <br> It currently supports both Singleplayer and Multiplayer.

> [!WARNING]
> This is not meant to be a pirated version of Minecraft, it is simply just some patches for Minecraft Alpha 1.2.6 that allows it to be playable in the browser. If you haven't already, please go [Buy Minecraft](https://www.minecraft.net/en-us/store/minecraft-java-bedrock-edition-pc?tabs=%7B%22details%22%3A0%7D).

## Quick Start
### Play Now:
**Client:** https://eaglerports.github.io/a1.2.6/

**Other Minecraft Web Ports:** https://eaglerports.github.io

### Offline Downloads:
You can grab the latest build here: [Releases](https://github.com/EaglerPorts/a1.2.6/releases) or compile it yourself.

## Compiling
Run the `CompileClient` script to build the JavaScript and WASM-GC clients. The results will be in the `client` folder.
